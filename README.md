# Dental-X-Rays-Caries-Segmentation
# Automated Dental Caries Segmentation and Panel Detection

This project provides an end-to-end, fully automated system for processing dental Bitewing X-ray images. It integrates deep learning and computer vision techniques to segment dental caries (tooth decay) and automatically recognize structured image panels, offering a fast and efficient pipeline for dental informatics.

---

## Key Features

* **Automated Panel Detection**: A newly integrated module that automatically identifies and segments the distinct multi-panel views (e.g., three-panel views) within a single radiographic image.
* **High-Speed Inference**: Optimized batch processing script to run inference on large datasets sequentially without manual intervention.
* **Caries Segmentation**: Leverages an EfficientNet-B0 backed U-Net model to generate precise binary masks for affected areas.
* **Zero-Configuration Pipeline**: Eliminates the need for Google Drive mounting inside Colab by using direct programmatic downloads, facilitating a standalone deployment.

---

## Pipeline Overview

The system operates as a fully automated pipeline:
1. **Input**: Raw dental X-ray images are loaded from a structured directory.
2. **Preprocessing**: Images are automatically resized and converted to grayscale to match the model's training distribution.
3. **Inference & Mask Generation**: The network processes the input to locate both the panel structures and caries, outputting precise binary masks.
